🎯 등차수열 문제 풀이 전략

주어진 조건에서 핵심 단서를 찾아 논리적으로 정답을 추론하는 과정

등차수열과 등비수열 1번 문제

Step 1: 조건 분석 - 숨겨진 관계 찾기

조건 (나)와 (다)의 차이를 이용해 음수 항들의 합을 구합니다.

• 핵심 관찰: 공차가 양수이고 \(a_k=0\)이므로, \(a_1, a_2, ..., a_{k-1}\) 항들은 음수입니다.

• 식 세우기: 조건 (다)에서 (나)를 빼면, \[ \sum_{i=1}^{m} |a_i| - \sum_{i=1}^{m} a_i = 84 - 60 = 24 \] \(i \ge k\) 일 때 \(a_i \ge 0\)이므로 \(|a_i| = a_i\)입니다. 따라서 식은 음수 항들만 남게 됩니다. \[ \sum_{i=1}^{k-1} |a_i| - \sum_{i=1}^{k-1} a_i = 24 \] \(a_i < 0\) 이므로 \(|a_i| = -a_i\) 입니다. \[ \sum_{i=1}^{k-1} (-a_i) - \sum_{i=1}^{k-1} a_i = -2 \sum_{i=1}^{k-1} a_i = 24 \]

→ 따라서 \(a_1\)부터 \(a_{k-1}\)까지의 합은 \(\sum_{i=1}^{k-1} a_i = -12\) 입니다.

Step 2: \(k\) 값 구하기 - 등차수열의 합 공식 활용

Step 1에서 구한 합의 정보를 이용해 \(k\) 값을 계산합니다.

• 항 표현하기: 공차 \(d=2\)이므로 \(a_{k-1} = a_k - d = 0 - 2 = -2\) 입니다. 또한 \(a_k = a_1 + (k-1)d = 0\) 에서 \(a_1 = -2(k-1)\) 입니다.

• 등차수열 합 공식 적용: 첫째항부터 \((k-1)\)항까지의 합 \(S_{k-1}\)은, \[ S_{k-1} = \frac{(k-1)(a_1 + a_{k-1})}{2} = -12 \] 위에서 구한 \(a_1\)과 \(a_{k-1}\)을 대입하면, \[ \frac{(k-1)(-2(k-1) - 2)}{2} = -12 \] \[ (k-1)(-k+1-1) = -12 \implies (k-1)(-k) = -12 \implies k(k-1) = 12 \]

✔️ \(k\)는 자연수이므로 \(k=4\) 입니다.

Step 3: 일반항 특정 및 \(m\) 값 구하기

\(k\)값을 이용해 일반항을 완성하고, 조건 (나)를 이용해 \(m\)을 구합니다.

  • 1️⃣ 일반항 \(a_n\) 구하기:

    \(k=4\)이므로 첫째항은 \(a_1 = -2(4-1) = -6\) 입니다. 일반항은 \(a_n = -6 + (n-1)2 = 2n - 8\) 입니다.

  • 2️⃣ \(m\) 값 계산하기:

    조건 (나)에서 \(S_m = 60\) 이므로 등차수열 합 공식을 적용합니다. \[ S_m = \frac{m(2a_1 + (m-1)d)}{2} = \frac{m(2(-6) + (m-1)2)}{2} = 60 \] \[ m(-12 + 2m - 2) = 120 \implies m(2m - 14) = 120 \implies m^2 - 7m - 60 = 0 \] 이차방정식을 풀면 \((m-12)(m+5)=0\) 입니다.

  • 🔑 \(m\)은 자연수이므로 \(m=12\) 입니다.

Step 4: 최종 답 구하기

찾아낸 \(k\)와 \(m\)의 값을 더하여 최종 답을 구합니다.

• 구한 값: \(k=4\), \(m=12\)

• 최종 계산: \(k+m = 4+12 = 16\)

최종 정답: \( 16 \)
