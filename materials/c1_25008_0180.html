수열의 합과 수학적 귀납법 1번 문제

Step 1: \(b_n\) 관계식 간단히 하기

등차수열 \(\{a_n\}\)의 공차가 \(-3\)임을 이용하여 \(b_n\)을 \(a_n\)에 대한 식으로 정리합니다.

• 주어진 식: \(b_n = 1 - \frac{a_{n+1}}{a_n}\)

• 등차수열 조건 적용: \(a_{n+1} = a_n - 3\)

• 식 대입 및 정리: \[b_n = 1 - \frac{a_n - 3}{a_n} = \frac{a_n - (a_n - 3)}{a_n} = \frac{3}{a_n}\]

→ 핵심 관계식: \(b_n = \frac{3}{a_n}\)

Step 2: 부분분수 분해 적용하기

합을 구하기 위해 \(b_k b_{k+1}\)을 부분분수 형태로 변환합니다.

• 합계 항 변환: \(b_k b_{k+1} = \frac{3}{a_k} \cdot \frac{3}{a_{k+1}} = \frac{9}{a_k a_{k+1}}\)

• 부분분수 공식: \(\frac{C}{AB} = \frac{C}{B-A} \left(\frac{1}{A} - \frac{1}{B}\right)\)

• 공식 적용: \(a_{k+1} - a_k = -3\) 이므로, \[\frac{9}{a_k a_{k+1}} = \frac{9}{-3} \left(\frac{1}{a_k} - \frac{1}{a_{k+1}}\right) = -3 \left(\frac{1}{a_k} - \frac{1}{a_{k+1}}\right)\]

Step 3: 망원급수(Telescoping Sum) 계산하기

부분분수로 변환된 수열의 합을 구하면, 중간 항들이 소거됩니다.

• 합계 식: \(\sum_{k=1}^{10} -3 \left(\frac{1}{a_k} - \frac{1}{a_{k+1}}\right) = -3 \sum_{k=1}^{10} \left(\frac{1}{a_k} - \frac{1}{a_{k+1}}\right)\)

• 전개 및 소거: \[-3 \left[ \left(\frac{1}{a_1} - \frac{1}{a_2}\right) + \left(\frac{1}{a_2} - \frac{1}{a_3}\right) + \dots + \left(\frac{1}{a_{10}} - \frac{1}{a_{11}}\right) \right]\]

🔑 계산 결과: \(-3 \left(\frac{1}{a_1} - \frac{1}{a_{11}}\right)\)

Step 4: \(a_1\)에 대한 방정식 풀이

계산 결과와 문제에 주어진 값을 이용하여 \(a_1\)에 대한 방정식을 세우고 풉니다.

1. 방정식 세우기: \(-3 \left(\frac{1}{a_1} - \frac{1}{a_{11}}\right) = -\frac{45}{28}\)

2. \(a_{11}\) 변환: \(a_{11} = a_1 + 10d = a_1 - 30\)

3. 방정식 정리: \[\frac{1}{a_1} - \frac{1}{a_1 - 30} = \frac{15}{28} \implies \frac{(a_1 - 30) - a_1}{a_1(a_1 - 30)} = \frac{15}{28}\] \[\frac{-30}{a_1^2 - 30a_1} = \frac{15}{28} \implies -56 = a_1^2 - 30a_1\]

4. 이차방정식 풀이: \[a_1^2 - 30a_1 + 56 = 0 \implies (a_1 - 2)(a_1 - 28) = 0\] \[\therefore a_1 = 2 \text{ 또는 } a_1 = 28\]

모든 가능한 \(a_1\)의 값의 합은?

\(2 + 28 = 30\)

복잡한 수열의 합 문제를 단계적으로 해결했습니다!
