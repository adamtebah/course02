수열의 합과 수학적 귀납법 2번 문제

Step 1: 함수 분석 및 \(p, q\) 값 찾기

함수 \(f(x) = p \sin q(x-r)\)의 주기와 함숫값 조건을 활용합니다.

• 주기 조건: 주기가 4이므로 \(\frac{2\pi}{q} = 4\). 따라서 \(q = \frac{\pi}{2}\) 입니다.

• 함숫값 조건 \(f(1)=1\): \(p \sin\left(\frac{\pi}{2}(1-r)\right) = p \cos\left(\frac{\pi}{2}r\right) = 1\)

• 함숫값 조건 \(f(2)=1\): \(p \sin\left(\frac{\pi}{2}(2-r)\right) = p \sin\left(\pi - \frac{\pi}{2}r\right) = p \sin\left(\frac{\pi}{2}r\right) = 1\)

• \(p\) 값 계산: \(\cos\left(\frac{\pi}{2}r\right) = \frac{1}{p}\), \(\sin\left(\frac{\pi}{2}r\right) = \frac{1}{p}\) 이므로, 삼각함수 제곱 관계 \(\sin^2\theta + \cos^2\theta = 1\)에 대입하면 \(\left(\frac{1}{p}\right)^2 + \left(\frac{1}{p}\right)^2 = 1\). 따라서 \(p^2=2\) 이고, \(p=\sqrt{2}\) 입니다.

Step 2: 수열 \(\{a_k\}\)의 규칙성 발견 및 합 계산

\(p=\sqrt{2}\)를 이용해 \(r\)의 값들을 찾고, 수열 \(\{a_k\}\)의 합을 구합니다.

• \(r\) 값 조건: \(\cos\left(\frac{\pi}{2}r\right) = \frac{1}{\sqrt{2}}\) 이고 \(\sin\left(\frac{\pi}{2}r\right) = \frac{1}{\sqrt{2}}\)를 동시에 만족해야 합니다.

• 일반해: 이를 만족하는 각은 \(\frac{\pi}{2}r = \frac{\pi}{4} + 2n\pi\) (단, \(n\)은 0 이상의 정수) 형태입니다.

• \(r\)의 일반항: 위 식을 \(r\)에 대해 정리하면 \(r = \frac{1}{2} + 4n\). 이것이 수열 \(\{a_k\}\)의 일반항이 됩니다. (\(a_k = \frac{1}{2} + 4(k-1)\))

• 수열 \(\{a_k\}\)의 합: 첫째항이 \(\frac{1}{2}\)이고 공차가 4인 등차수열의 합입니다. \[ \sum_{k=1}^{6} a_k = \frac{6 \left( 2 \times \frac{1}{2} + (6-1) \times 4 \right)}{2} = 3(1 + 20) = 63 \]

Step 3: 수열 \(\{f(k)\}\)의 규칙성 발견 및 합 계산

함수 \(f(x)\)의 주기가 4임을 이용하여 \(\{f(k)\}\)의 값들을 찾고 합을 구합니다.

• 주기성 활용: \(f(x)\)의 주기가 4이므로, \(f(k)\)의 값은 4를 주기로 반복됩니다.

• 함숫값 나열: \(f(1)=1, f(2)=1\). 주기에 따라 \(f(3)=f(2+1)=-1\), \(f(4)=f(2+2)=-1\), \(f(5)=f(1)=1\), \(f(6)=f(2)=1\).

• 수열 \(\{f(k)\}\)의 합: \[ \sum_{k=1}^{6} f(k) = f(1) + f(2) + f(3) + f(4) + f(5) + f(6) \] \[ = 1 + 1 + (-1) + (-1) + 1 + 1 = 2 \]

Step 4: 최종 답 구하기

두 시그마 값을 더하여 최종 답을 도출합니다.

• 최종 계산: \[ \sum_{k=1}^{6} f(k) + \sum_{k=1}^{6} a_k = 2 + 63 = 65 \]

최종 정답: \( 65 \)
