지수와 로그 1번 문제

문제의 핵심 정의

2 이상의 자연수 n에 대하여 실수 a의 n제곱근 중 실수인 것의 개수를 fn(a)라 하자.

Step 1: 목표 설정 - 최댓값 찾기

주어진 식 fn(an+1) + fn(fn+1(a))의 값이 최대일 때를 찾아야 합니다.

핵심 아이디어: fn(a)의 값은 0, 1, 2만 가능하므로, 최댓값은 2 + 2 = 4가 될 때입니다.

Step 2: 조건 분석 - 언제 2가 될까?

fn(x) = 2가 되기 위한 조건을 떠올립니다.

✔️ 조건: n은 짝수, x는 양수

Step 3: 핵심 단서 추론

최댓값 조건(f=2)을 각 항에 적용하여 n과 a의 조건을 구합니다.

  • 1️⃣ 첫 번째 항: fn(an+1) = 2

    → n은 짝수이고, an+1 > 0 이어야 합니다.

    → n이 짝수이므로 n+1은 홀수. 따라서 a > 0 이어야 합니다.

  • 2️⃣ 두 번째 항: fn(fn+1(a)) = 2

    → 괄호 안부터 계산: n+1은 홀수, a > 0이므로 fn+1(a) = 1 입니다.

    → 즉, fn(1) = 2. n은 짝수, 1은 양수이므로 조건에 부합합니다.

최종 조건 확정!

최댓값을 가질 조건은 n은 짝수, a > 0 입니다.

Step 5: 최종 답 구하기

확정된 조건(n: 짝수, a > 0)을 사용하여 아래 식의 값을 계산합니다.

fn(-a) + fn+1(a) + fn+2(|a|-a)

• fn(-a): n은 짝수, -a는 음수 → 0

• fn+1(a): n+1은 홀수, a는 양수 → 1

• fn+2(|a|-a): a>0이므로 |a|-a=0. 즉 fn+2(0) → 1

최종 정답: 0 + 1 + 1 = 2

복잡한 경우의 수를 나누는 대신, 문제의 핵심 조건을 먼저 찾아 해결했습니다!
