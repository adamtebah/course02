🎯 삼각방정식 실근 개수 문제 완전 정복

치환과 근의 분리를 이용한 전략적 문제 해결

삼각함수 3번 문제

Step 1: 각 변환 공식으로 식 다듬기

복잡한 삼각함수 항들을 간단한 형태로 변환합니다.

• \(\cos\left(\frac{\pi}{2} - x\right) \rightarrow \sin x\)

• \(\sin(\pi + x) \rightarrow -\sin x\)

변환된 식: \(2\sin^2 x - 3a\sin x + a(-\sin x) - a + 8 = 0\)

Step 2: \(\sin x = t\) 로 치환하기

방정식을 다루기 쉬운 이차방정식 형태로 만듭니다.

✔️ 치환 및 정리

\(2\sin^2 x - 4a\sin x - a + 8 = 0\)

⬇️

\(2t^2 - 4at - a + 8 = 0\)

✔️ 치환 시 가장 중요한 것: 범위 확인!

문제 조건 \(0 \le x \le \pi\) 이므로, \(t = \sin x\)의 범위는 \(0 \le t \le 1\) 입니다.

Step 3: 실근의 의미 파악하기

문제의 목표는 \(x\)의 실근이 2개! \(t\)의 값에 따라 \(x\)의 개수가 어떻게 달라지는지 분석합니다.

• \(t\)값이 \(0 \le t < 1\) 범위에 있으면 → \(x\)는 2개

• \(t\)값이 \(t=1\) 이면 → \(x\)는 1개 (\(x=\frac{\pi}{2}\))

• \(t\)값이 \(t=0\) 이면 → \(x\)는 2개 (\(x=0, \pi\))

결론: \(x\)의 실근이 2개가 되려면, 이차방정식 \(2t^2 - 4at - a + 8 = 0\)은 \(0 \le t < 1\) 범위에서 오직 하나의 실근을 가져야 합니다.

Step 4: '근의 분리'로 경우 나누기

\(f(t) = 2t^2 - 4at - a + 8\)의 근이 \(0 \le t < 1\)에서 하나만 존재할 조건을 두 경우로 나눕니다.

  • Case 1: 한 근이 0과 1 사이에 있는 경우

    경계값의 부호가 달라야 합니다: \(f(0) \cdot f(1) < 0\)

    \(f(0) = -a + 8\)

    \(f(1) = 2 - 4a - a + 8 = -5a + 10\)

    \((-a+8)(-5a+10) < 0 \implies (a-8)(5a-10) < 0\)

    결과: \(2 < a < 8\)

  • Case 2: 한 근이 정확히 \(t=0\)인 경우

    \(f(0) = 0\) 이어야 합니다.

    \(-a + 8 = 0 \implies a = 8\)

    \(a=8\)일 때, \(2t^2 - 32t = 0 \implies 2t(t-16)=0\). 근은 \(t=0, t=16\).

    \(t=0\)은 범위에 포함, \(t=16\)은 범위 밖이므로 조건 만족!

    결과: \(a = 8\)

최종 결론

두 경우의 결과를 합치면, \(a\)의 범위는 \(2 < a \le 8\) 입니다.

해당 범위의 정수 \(a\)는 3, 4, 5, 6, 7, 8
