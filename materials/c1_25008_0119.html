숨겨진 조건을 찾아 여러 개념을 종합적으로 활용하기

사인법칙과 코사인법칙 1번 문제

문제의 핵심 조건

\(\triangle ABC\)에서 \(\angle A=90^\circ, \angle B=60^\circ, \overline{BC}=2\) 이고, 내부의 점 P에 대하여 \(\angle PAB = \angle PBC = \angle PCA = \theta\) 일 때, \(\overline{AP}^2 + \overline{BP}^2\)의 값을 구하는 문제입니다.

Step 1: 기본 정보 파악

주어진 조건으로 \(\triangle ABC\)가 \(30^\circ-60^\circ-90^\circ\) 특수각 삼각형임을 알 수 있습니다. 이를 통해 모든 변의 길이를 먼저 구합니다.

✔️ 결과: \(\overline{AB}=1, \overline{AC}=\sqrt{3}, \overline{BC}=2\)

Step 2: 사인법칙으로 \(\overline{CP}\) 표현하기 (1)

\(\triangle PBC\)에 사인법칙을 적용하여 \(\overline{CP}\)를 \(\theta\)에 대한 식으로 나타냅니다. 먼저 \(\angle BPC = 180^\circ - \angle PBC - \angle BCP = 180^\circ - \theta - (30^\circ - \theta) = 150^\circ\) 입니다.

✔️ 식 정리: \(\frac{\overline{CP}}{\sin\theta} = \frac{\overline{BC}}{\sin(150^\circ)}\) 이므로, \(\overline{CP} = \frac{2\sin\theta}{1/2} = 4\sin\theta\)

Step 3: 결정적 단서 찾기 및 \(\overline{CP}\) 표현하기 (2)

\(\triangle PCA\)에 주목합니다. \(\angle APC = 180^\circ - \angle PAC - \angle PCA = 180^\circ - (90^\circ-\theta) - \theta = 90^\circ\) 임을 발견하는 것이 이 문제의 핵심입니다. 이 특수조건을 이용해 \(\triangle PCA\)에 사인법칙을 다시 적용합니다.

핵심 아이디어: \(\frac{\overline{CP}}{\sin(90^\circ-\theta)} = \frac{\overline{AC}}{\sin(90^\circ)}\) 이므로, \(\overline{CP} = \overline{AC} \cos\theta = \sqrt{3}\cos\theta\) 입니다.

Step 4: \(\tan\theta\) 값 계산하기

Step 2와 3에서 구한 \(\overline{CP}\)에 대한 두 식은 서로 같으므로, \(4\sin\theta = \sqrt{3}\cos\theta\) 입니다.

✔️ 결과: \(\tan\theta = \frac{\sin\theta}{\cos\theta} = \frac{\sqrt{3}}{4}\)

Step 5: 최종 답 구하기

  1. \(\tan\theta = \frac{\sqrt{3}}{4}\) 값을 이용해 직각삼각형(밑변 4, 높이 \(\sqrt{3}\))을 그리면, 빗변은 \(\sqrt{4^2 + (\sqrt{3})^2} = \sqrt{19}\) 입니다. 따라서 \(\sin^2\theta = \left(\frac{\sqrt{3}}{\sqrt{19}}\right)^2 = \frac{3}{19}\) 입니다.
  2. \(\overline{AP}\)와 \(\overline{BP}\)를 \(\sin\theta\)로 표현하여 제곱합니다.

    \(\triangle PCA\)가 직각삼각형이므로, \(\overline{AP} = \overline{AC}\sin\theta = \sqrt{3}\sin\theta\).
    따라서 \(\overline{AP}^2 = 3\sin^2\theta = 3 \times \frac{3}{19} = \frac{9}{19}\)

    \(\triangle PAB\)에서 \(\overline{BP} = \frac{\overline{AB}\sin\theta}{\sin(120^\circ)} = \frac{1 \cdot \sin\theta}{\sqrt{3}/2} = \frac{2}{\sqrt{3}}\sin\theta\).
    따라서 \(\overline{BP}^2 = \frac{4}{3}\sin^2\theta = \frac{4}{3} \times \frac{3}{19} = \frac{4}{19}\)

  3. 두 값을 더하여 최종 답을 구합니다.

    \(\overline{AP}^2 + \overline{BP}^2 = \frac{9}{19} + \frac{4}{19} = \frac{13}{19}\)

최종 정답: \[\frac{13}{19}\]

사인법칙과 숨겨진 직각삼각형의 성질을 조합하여 문제를 해결했습니다!
