🎯 삼각함수 최대/최소 문제 완전 정복

복잡한 함수의 핵심을 파악하여 최적의 경로로 문제 해결하기

삼각함수 2번 문제

문제의 핵심 정의

\(f(x)=2\cos\frac{x}{k}\)의 주기를 \(g(k)\)라 할 때,

함수 \(h(k) = f(g(k)) - \sin\left(\frac{g(k)}{9} + \frac{7}{6}\pi\right)\)의 최대/최소를 분석해야 합니다.

Step 1: 함수 단순화 - 문제의 본질 파악하기

\(g(k)\)는 \(f(x)\)의 주기이므로 \(g(k) = 2\pi k\) 입니다.

이를 \(f(g(k))\)에 대입하면 놀랍도록 간단해집니다.

🔑 핵심 발견:

\[f(g(k)) = 2\cos\left(\frac{2\pi k}{k}\right) = 2\cos(2\pi) = 2 \times 1 = 2\]

따라서, \[h(k) = 2 - \sin\left(\frac{2\pi k}{9} + \frac{7}{6}\pi\right)\]

Step 2: 최대/최소 조건 분석 - 사인(sin)의 역할

\(h(k) = 2 - (\text{사인 값})\) 이므로, 덧셈/뺄셈 관계를 생각합니다.

  • ✔️ 최댓값을 가지려면? → 빼는 \(\sin\) 값이 가장 작아야 한다. (\(\sin(\theta) = -1\))
  • ✔️ 최솟값을 가지려면? → 빼는 \(\sin\) 값이 가장 커야 한다. (\(\sin(\theta) = 1\))

Step 3 & 4: 방정식 풀이로 \(a, b\) 찾기

위 조건을 만족하는 가장 작은 양수 \(k\)를 구합니다. (\(n\)은 정수)

1️⃣ 최댓값 조건 (\(a\) 구하기): \(\sin(\theta) = -1 \implies \theta = \frac{3\pi}{2} + 2n\pi\)

\[\frac{2\pi k}{9} + \frac{7}{6}\pi = \frac{3\pi}{2} + 2n\pi\]

\[\implies \frac{2\pi k}{9} = \frac{\pi}{3} + 2n\pi \implies k = \frac{3}{2} + 9n\]

→ 가장 작은 양수 \(k\)는 \(n=0\)일 때, \(a = \frac{3}{2}\)

2️⃣ 최솟값 조건 (\(b\) 구하기): \(\sin(\theta) = 1 \implies \theta = \frac{\pi}{2} + 2n\pi\)

\[\frac{2\pi k}{9} + \frac{7}{6}\pi = \frac{\pi}{2} + 2n\pi\]

\[\implies \frac{2\pi k}{9} = -\frac{2\pi}{3} + 2n\pi \implies k = -3 + 9n\]

→ 가장 작은 양수 \(k\)는 \(n=1\)일 때, \(b = 6\)

Step 5: 최종 답 구하기

이제 구한 \(a\)와 \(b\)의 값을 나누어주면 끝!

\[\frac{a}{b} = \frac{3/2}{6} = \frac{3}{12} = \frac{1}{4}\]

최종 정답: ② 1/4
