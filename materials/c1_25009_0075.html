🎯 미분계수와 극한 심화 문제 풀이 전략

곱의 미분법과 극한의 성질을 활용한 단계별 분석

미분계수와 도함수 1번 문제

Step 1: 목표 설정 - 문제의 본질 파악하기

주어진 극한식은 \(h(x) = f(x)g(x)\)로 둘 때, \(h'(1)\)의 미분계수 정의와 일치합니다.

• 주어진 식: \( \lim_{x \to 1} \frac{f(x)g(x) - f(1)g(1)}{x-1} = h'(1) \)

• 곱의 미분법 적용: \(h'(1) = f'(1)g(1) + f(1)g'(1)\)

• 최종 목표: \(f(1), g(1), f'(1), g'(1)\) 네 개의 값을 구하여 \(h'(1)\)을 계산합니다.

Step 2: 조건 (가), (나) 분석하기

극한의 성질(\(\frac{0}{0}\) 꼴)과 미분계수의 정의를 이용하여 관계식을 도출합니다.

조건 (가) 분석

\( \lim_{x \to 1} \frac{3f(x) - g(1)}{x-1} = g(1) + 3 \)

1. 분모 \(\to 0\)이므로 분자 \(\to 0\)
\(\implies 3f(1) - g(1) = 0 \implies \boldsymbol{g(1) = 3f(1)}\)

2. \(g(1)\)을 대입하여 미분계수 형태로 변환
\(\implies 3f'(1) = g(1) + 3\)

조건 (나) 분석

\( \lim_{x \to 1} \frac{g(x) - 3f(1)}{x-1} = f(1) + 11 \)

1. \(3f(1)\)을 \(g(1)\)으로 치환
\(\implies \lim_{x \to 1} \frac{g(x) - g(1)}{x-1} = g'(1)\)

2. 미분계수와 우변을 연결
\(\implies g'(1) = f(1) + 11\)

Step 3: 마지막 단서 찾기

문제의 첫 번째 식 \(g(x) = (2x+1)f(x)\)를 미분하여 네 번째 관계식을 완성합니다.

• 원본 식: \(g(x) = (2x+1)f(x)\)

• 양변 미분 (곱의 미분법): \(g'(x) = 2f(x) + (2x+1)f'(x)\)

• \(x=1\) 대입: \(g'(1) = 2f(1) + 3f'(1)\)

Step 4: 연립방정식 풀이

지금까지 구한 네 개의 식을 연립하여 각 값을 계산합니다.

1. \(3f'(1) = 3f(1) + 3 \implies \boldsymbol{f'(1) = f(1) + 1}\)

2. \(g'(1) = 2f(1) + 3(f(1)+1) \implies \boldsymbol{g'(1) = 5f(1) + 3}\)

3. \(5f(1) + 3 = f(1) + 11 \implies 4f(1) = 8 \implies \boldsymbol{f(1) = 2}\)

4. \(\boldsymbol{g(1) = 3(2) = 6}\), \(\boldsymbol{f'(1) = 2 + 1 = 3}\), \(\boldsymbol{g'(1) = 2 + 11 = 13}\)

Step 5: 최종 답 계산

Step 1에서 설정한 목표 식에 계산된 값들을 대입합니다.

\(h'(1) = f'(1)g(1) + f(1)g'(1)\)

\(= (3)(6) + (2)(13)\)

\(= 18 + 26 = \boldsymbol{44}\)

최종 정답: \( 44 \)
